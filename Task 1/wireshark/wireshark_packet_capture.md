# ðŸŸ¦ 1. Starting Wireshark Capture (Windows)

### Step-by-step

1. Open **Wireshark** (Run as Administrator recommended).
2. Select your active interface:
   - `Wi-Fi` (for wireless connections)
   - `Ethernet` (for wired)
3. Click **Start Capturing Packets** (blue shark fin icon).
4. Keep Wireshark running in the background while performing the Nmap scan.

You should see packets flowing in real-time.

# ðŸŸ¦ 3. Apply Important Filters

To properly analyze the scan, apply these filters in Wiresharkâ€™s filter bar.

## A. View outgoing SYN packets (scan attempts)

```
tcp.flags.syn == 1 && tcp.flags.ack == 0
```

These are packets your PC sends to check if a port is open.

## B. View SYN/ACK responses (open ports)

```
tcp.flags.syn == 1 && tcp.flags.ack == 1
```

These responses indicate **open ports**.

## C. View RST responses (closed ports)

```
tcp.flags.rst == 1
```

This indicates the port is **closed**.

## D. Only show traffic between you and a specific host

```
ip.addr == 192.168.223.243
```

## E. Only TCP traffic

```
tcp
```

These filters help isolate Nmap traffic clearly.
